Nombre del proyecto:
Modelo para predecir precio de Autos.

Objetivo:
Desarrollar e implementar un modelo de machine learning que sea capaz de predecir el precio para vender un vehículo usado,
tomando en cuenta sus caracteristicas como antiguedad, kilometraje, motor, marca, modelo, etc.
